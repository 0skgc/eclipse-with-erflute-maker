# eclipse-with-erflute-maker

## このスクリプトは以下のことを実行します

1. eclipse-oxygen(Java)をダウンロード
2. ERFluteプラグインのjarをダウンロード
3. eclipseのdmgパッケージを展開し、ERFluteプラグインおよびそれが稼働するために必要なプラグインをインストール

## 展開されたeclipseを実行する方法

open ./target/Eclipse.app
